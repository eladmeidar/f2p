f2p - a FriendFeed application for mobile

'f2p' is a web-based FriendFeed application for mobile users by NaHi. It's based on FriendFeed API but it's not formally affiliated with FriendFeed. Let me know about any trouble on this service at http://friendfeed.com/f2p.
